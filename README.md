# What is HexaForge?

HexaForge is a tank battle game where the player fights aganist the computer. The main goal is to destroy the enemy tank before it destroys yours. Both tanks can move, aim, and fire shells that deal damage depending on the shot.

## HexaForge in Python

This game was built using the **Pygame** module. The player controls the tank using the keyboard with the following controls:

* Right and Left arrows to move the tank.
* Up and Down arrows to move the turret.
* **A** and **D** keys to decrease or increase the fire power.
* **Space** to fire.

## Project Prerequisites

Before running the game, make sure you have **Pygame** installed.

```bash
pip install pygame
```

## Steps to build HexaForge

We followed these steps while making the game:

1. First, we imported the required modules.
2. Then we initialized pygame and created the global variables.
3. After that, we added functions to draw text and game information on the screen.
4. Next, we created the button class for the menus.
5. Then we made the main menu window.
6. After that, we created the controls screen.
7. Next, we wrote the function that draws the tanks.
8. Then we added the explosion effect.
9. After that, we created the firing system for both the player and the computer.
10. We also added pause and game over screens.
11. Finaly, we created the main game loop that runs everything togther.
